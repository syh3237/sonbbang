# 프로젝트 개요

## 프로젝트명

Schedule Share

## 프로젝트 소개

사용자 간 친구 연결을 통해 일정을 공유할 수 있는 캘린더 기반 서비스

웹 서비스를 먼저 개발한 후 Flutter 앱으로 확장한다.

---

## 기술 스택

### Frontend

- Vue3
- Vite
- Vue Router
- Pinia
- SCSS

### Backend

- Supabase

### Mobile

- Flutter (예정)

---

## 주요 기능

### 1차 개발

- 회원가입
- 로그인
- 친구 추가
- 친구 수락 / 거절
- 캘린더 조회
- 일정 등록
- 일정 수정
- 일정 삭제
- 일정 공유

### 2차 개발

- 가계부 기능
- 수입 / 지출 관리
- 월별 통계
- 카테고리 관리

---

## 서비스 정책

- 사용자는 여러 명의 친구를 등록할 수 있다.
- 친구로 연결된 사용자끼리 일정을 공유할 수 있다.
- 공개 범위에 따라 일정 노출 여부를 결정한다.

---

## 개발 원칙

- UI와 데이터 로직을 분리한다.
- 컴포넌트에서 직접 Supabase 호출 금지
- services 레이어를 통해 데이터 접근
- 시맨틱 마크업 우선
- 웹 접근성 준수
